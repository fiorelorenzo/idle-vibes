# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com),
and this project adheres to [Semantic Versioning](https://semver.org).

## [0.1.0] - 2026-04-13

### Added

- Monorepo structure with shared types, VS Code extension, and Svelte UI packages
- Smart Parser with three-pillar detection system:
  - Pillar A: AI token detection (80+ chars/sec threshold)
  - Pillar B: Diagnostics delta on save
  - Pillar C: AST heuristics and Git commit analysis
- Vibe System with four states (Dead Zone, Cruising, Flow State, Peak Vibe)
- Vibe Meter with decay, pressure, and Flow State Events
- Three-tier resource economy (Volatile, Stable, Persistent)
- Colony management: 4 Proxy roles (Miner, Constructor, Guard, Operator)
- 10 building types with conversion, defense, and vibe infrastructure
- Mining map with 5 depth layers (Surface through Abyss)
- Glitch system with 4 enemy types tied to code quality
- Codex with 6 categories and progressive unlock system
- Prestige system (Commit & Push) with Tech Tree
- ARIA narrative system with 3 lore entries
- PixiJS colony renderer (30fps cap)
- Standby mode with passive generation and scheduled jobs
- Supabase schema with RLS policies
- Extension points: Parser plugins, Codex entries, Vibe hooks
